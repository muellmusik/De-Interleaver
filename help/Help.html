De-Interleaver


This is a simple application that does only two things: Take a multichannel audio file and split it into mono files, or the reverse.


Input formats are anything that libsndfile reads (so almost anything!). Check here for details: http://www.mega-nerd.com/libsndfile/


Output formats are aiff, wav, or caf (64-bit CoreAudio format, which allows for really large interleaved files!). The output sample type will be that of the first input file, providing it's supported by the output format. If not it will default to 32 bit floating point, which in most cases should mean no loss of resolution.


Interleaving


For interleaving you can have files of different lengths, and the application will simply zero pad the end of the shorter files. Other than that nothing much fancy, you can't interleave multichannel files (sorry, split them into mono and then do it) and there's no mixing down or anything like that.


You can change the channel order by dragging files in the table.


It assumes you know what you're doing, and doesn't prohibit interleaving files with different sample rates (output sr is that of the first channel) or types.


Deinterleaving


You can edit the names of the output channels by double-clicking on the table cells.
